# Yutbre - YouTube Video & Audio Downloader

A modern, lightweight YouTube downloader built with Python and CustomTkinter.

![YouTube Downloader Preview](preview.png)

## Features

- Download YouTube videos in high quality
- Extract audio only (MP3 format recommended with additional tools)
- Clean, modern dark-themed interface
- Progress tracking
- Custom download location selection

## Requirements

- Python 3.8+
- customtkinter
- pytubefix

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/yutbre.git
cd yutbre
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

## Usage

1. Enter a valid YouTube video URL
2. Select your download folder using the Browse button
3. Choose between Video or Audio download
4. Click Download and wait for completion

## Tech Stack

- **Python** - Core programming language
- **CustomTkinter** - Modern GUI framework
- **PyTubeFix** - YouTube video downloading
